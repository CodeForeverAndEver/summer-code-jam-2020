- About Coding Arena -

Coding Arena is an online coding battle where two users
compete to finish a python coding problem the fastest
in a limited timeframe.



Coding Arena was written with Django, Django Channels,
Python and HTML5. Our group also used Docker, which was
used to run the user's code.


Our group also implemented a chat room in the main
"competiton page", in which users can talk to each other
while they brawl.
